## [단축 버전] (800자 이내, 제안서 기대효과 항목 내 삽입용)

**유학생 금융 가치:** 2025년 국내 유학생 253,434명, 65.5%가 졸업 후 한국 체류를 계획한다. 유학생 1인당 연간 금융 거래 추정액은 약 2,000만 원(등록금 800만, 생활비 700만, 송금 300만, 기타 200만)이며, 하나은행은 외국인 고객 737만 중 316만(43%)을 보유해 외국인 세그먼트 1위다. 유학생을 입국 초기에 온보딩하면, 졸업→취업→정주로 이어지는 10년 이상의 장기 고객 파이프라인을 확보할 수 있다.

**하나금융 사업 가치:** (1) 초기 온보딩 성공률 향상으로 유학생 이탈 방지, (2) 상담/지점 반복 방문 비용 절감(건당 약 8,000~15,000원 추정), (3) 외국인 세그먼트 점유율 확대 및 교차판매(카드·보험·송금) 기반 구축.

**확장 로드맵:** Phase 1 유학생(D-2) → Phase 2 어학연수(D-4)·워킹홀리데이(H-1) → Phase 3 취업자(E-2, E-7)·결혼이민(F-6). 동일 AI 엔진에 비자별 체크리스트만 교체하는 구조로, 확장 시 추가 개발 비용이 최소화된다.

### 2. 하나금융 사업 가치

#### 2-1. 고객 이탈 방지 (초기 온보딩 → 장기 고객 파이프라인)

| 현재 문제 | AdroadEZ 적용 후 |
|-----------|-------------------|
| 계좌개설까지 평균 2~3회 방문, 중도 포기 발생 | 사전 안내로 **1회 방문 완료** 유도 |
| 온보딩 실패 시 타행으로 이탈 | 체크리스트 기반 **단계별 잠금해제**로 이탈 차단 |
| 졸업 후 급여이체 계좌를 타행에 개설 | 온보딩 시점부터 **장기 관계** 구축 → 전환 비용 발생 |

> **핵심 지표:** 유학생 온보딩 완료율 50% → 80% 향상 시,
> 연간 약 **40,000명 추가 확보** → 장기 LTV 기준 **수천억 원 규모** 잠재 가치

#### 2-2. 운영 비용 절감

| 비용 항목 | 현재 추정 | AdroadEZ 적용 후 |
|-----------|-----------|-------------------|
| 외국인 상담 건당 비용 | 약 8,000~15,000원 (통역·통신·시간) | AI 비서로 **60~70% 감소** |
| 지점 반복 방문 비용 | 1회 방문당 약 10,000~20,000원 (인건비 기준) | 사전 서류 안내로 **재방문 50% 감소** |
| 연간 유학생 관련 상담 추정 | 약 50만 건 (신규+기존) | 약 **15~20만 건으로 감소** |

> **예상 절감액:** 연간 약 **30~50억 원** 수준의 운영비 절감 효과

#### 2-3. 외국인 세그먼트 점유율 확대

- 현재 하나은행 외국인 점유율 **43%** (1위)
- AdroadEZ를 통한 유학생 선점 → 졸업 후 취업·정주까지 록인(lock-in) 효과
- 타행 대비 **차별화된 외국인 온보딩 경험** → 구전(word-of-mouth) 효과
  - 유학생 커뮤니티 특성상 **학교별·국적별 네트워크**를 통한 자연 확산
- **교차판매 기회:** 계좌 → 카드 → 송금 → 보험 → 대출 → 자산관리
  - 온보딩 완료 고객의 교차판매 전환율 **2~3배 향상** 기대

### 3. 확장 로드맵: 동일 엔진, 체크리스트만 교체

```

│                    AdroadEZ 공통 AI 엔진                         │

└──────────┬──────────────┬──────────────┬────────────────────────┘
           │              │              │
     ┌─────▼─────┐  ┌────▼─────┐  ┌────▼──────┐
     │  Phase 1   │  │ Phase 2  │  │  Phase 3  │
     │  유학생     │  │ 연수/WH  │  │ 취업/이민  │
     │  (D-2)     │  │(D-4,H-1) │  │(E-2,E-7,  │
     │            │  │          │  │ F-6)      │
     └────────────┘  └──────────┘  └───────────┘
```

>
> AdroadEZ는 하나은행의 기존 인프라 위에 '유학생 여정'을 연결하는 온보딩 레이어로,
