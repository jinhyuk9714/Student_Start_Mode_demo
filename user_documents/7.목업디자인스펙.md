## 2. 별벗 (Byulbeot) 캐릭터 통합 가이드

### 별벗이란

하나금융그룹의 공식 AI 캐릭터. 머리에 초록 새싹이 달린 귀여운 AI 로봇.
"하나패밀리와 늘 함께하는 AI 안드로이드 로봇, 디지털 기술을 바탕으로 하루가 다르게 발전하고 있습니다."

### 활용 원칙

| 원칙 | 설명 |
|------|------|
| **가이드, 주인공 아님** | 별벗은 유저를 돕는 가이드. 화면을 지배하지 않는다. Duolingo 올빼미 수준의 과잉 노출 X |
| **AI 영역에서만 등장** | 규칙 기반 체크리스트 본체에는 별벗 없음. AI 챗, 안내, 축하 등 "AI가 말하는" 맥락에서만 |
| **감정 표현** | 별벗의 표정으로 상황 맥락 전달. 축하(웃음), 대기(생각), 안내(손 흔들기), 주의(놀람) |
| **크기 계층** | 대: 80px (온보딩, 빈 화면), 중: 48px (챗 아바타, 알림), 소: 24px (인라인 힌트) |

### 화면별 별벗 등장 지점

```
┌─────────────────────────────────────────────────────────────┐
│ 화면              │ 별벗 역할          │ 크기  │ 표정        │
├───────────────────┼────────────────────┼───────┼────────────┤
│ 온보딩 설문       │ 환영 인사          │ 80px  │ 손 흔들기   │
│ 로딩 화면         │ 분석 중 애니메이션   │ 80px  │ 생각 중     │
│ 홈 - 첫 방문      │ "안녕! 같이 해보자" │ 48px  │ 웃음       │
│ 홈 - 재방문       │ 없음 (깔끔하게)     │ -     │ -          │
│ 체크리스트        │ 없음 (규칙 영역)    │ -     │ -          │
│ 체크리스트 빈상태  │ "아직 할 일이 없어" │ 80px  │ 편안       │
│ 체크리스트 완료    │ "다 했어! 대단해!"  │ 80px  │ 축하       │
│ 아이템 펼침 - AI  │ 챗 버튼 옆 아이콘   │ 24px  │ 기본       │
│ 챗 탭            │ 메시지 아바타       │ 48px  │ 상황별     │
│ 챗 - 확인 카드    │ 카드 위 아이콘      │ 24px  │ 확인 질문   │
│ 마이 탭          │ 없음               │ -     │ -          │
│ 에러 화면        │ "잠시 문제가 생겼어" │ 80px  │ 놀람       │
│ cascade unlock   │ "새로운 게 열렸어!" │ 48px  │ 기쁨       │
│ NEW 항목 추가    │ "새 할일이 생겼어"  │ 48px  │ 안내       │
└─────────────────────────────────────────────────────────────┘
```

### AI Slope 경계와 별벗

```
규칙 엔진 영역 (별벗 없음)        AI 영역 (별벗 등장)
───────────────────────         ───────────────────────
체크리스트 본문                  AI 챗 대화
항목 상태 (잠김/가능/완료)        확인 카드
불가 사유, 해제 조건              RAG 검색 답변
cascade unlock 로직              질문 안내
딥링크, 준비물 목록               상황 인식 ("ARC 받으셨나요?")

→ 유저가 자연스럽게 "별벗이 말하는 건 AI, 체크리스트 자체는 시스템"을 인식
```

**잠긴 항목 펼침 (탭 시):**

```
  ┌────────────────────────────┐
  │ 🔒 은행 계좌 개설       ▲   │
  │ ─────────────────────────  │
  │                            │
```

**디자인 포인트:**
- 아코디언 펼침: 300ms ease-out 애니메이션
- 상태 그룹핑: 완료 → 가능 → 잠김 순서로 자연스러운 흐름
- 펼침 내부 "이걸 하면" 섹션: cascade 프리뷰. 유저가 "이걸 하면 뭐가 열리는지" 즉시 파악
- 잠긴 항목도 펼침 가능 (투명성). AI 챗 버튼 제공 (잠겨 있어도 질문은 가능)
- 별벗은 체크리스트 본체에 없음. "별벗에게 물어보기" 버튼에만 24px 아이콘

## 5. 인터랙션 상세

### 확인 카드 (Confirmation Card) 컴포넌트

```
Props:
  item_code: string       ("OPEN_BANK_ACCOUNT")
  item_title: string      ("하나은행 계좌 개설")
  onConfirm: () => void   (POST /api/progress)
  onDismiss: () => void   (아무 일도 안 일어남)

Render:
  ┌──────────────────────────────┐
  │  ☑ {item_title}              │  ← 체크 아이콘 Primary색
  │                              │
  │  [✓ 완료 확인]  [아직 아니요]  │  ← Primary btn + Ghost btn
  └──────────────────────────────┘
  
  Background: #F8F9FA
  Border: 1px solid #E5E7EB
  Radius: 12px
  Padding: 16px
  
  ✓ 버튼: bg Primary, text white, radius 8px, h 40px
  아직 버튼: bg transparent, text Muted, radius 8px, h 40px
```

### 아코디언 펼침 (Accordion)

```
닫힘 → 열림: 300ms ease-out, max-height 애니메이션
열림 → 닫힘: 200ms ease-in
▼ 아이콘: 180도 회전 (닫힘→열림)
동시에 하나만 열림 (같은 그룹 내)
```

### 하단 탭바

```
높이: 56px + safe area inset
아이콘: 24x24px
텍스트: 11px
활성: Primary 색 (아이콘 + 텍스트)
비활성: Muted 색
탭 전환: 즉시 (애니메이션 없음, 빠른 반응)
```

### cascade unlock 토스트

```
체크리스트에서 항목 완료 시:
  하단에서 올라오는 토스트 (3초 자동 닫힘)
  "✨ 2개 항목이 새로 가능해졌어요!"
  [체크리스트 보기 →]
  
  Background: #1A1A2E (다크)
  Text: white
  Radius: 12px
  Width: calc(100% - 40px)
  Animation: slide-up 300ms
```

## 6. 다국어 고려사항

| 고려 | 대응 |
|------|------|
| 한글/영문 길이 차이 | 버튼, 라벨 min-width 설정. 텍스트 truncation 정책 |
| 중국어 간체 | Pretendard는 CJK 미지원 → Noto Sans SC fallback |
| 베트남어 성조부호 | line-height 1.5 이상으로 diacritics 공간 확보 |
| RTL 없음 | 대상 언어 모두 LTR이므로 RTL 대응 불필요 |
| 별벗 말풍선 | 번역 시 말풍선 길이 변화 → max-width 제한 + 자동 줄바꿈 |
